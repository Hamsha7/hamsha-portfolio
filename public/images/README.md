# Images Folder

## How to Add Your Photo

1. Place your photo in this folder (`public/images/`)
2. Name it: `hamsha-photo.jpeg` (or `.png`, `.webp`, `.jpg`)
3. Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`

## Recommended Image Specifications

- **Format**: JPG or PNG
- **Recommended size**: 800x1000px or higher (portrait orientation)
- **File size**: Under 500KB for optimal loading
- **Aspect ratio**: 2:3 or 3:4 (portrait) works best

## Current Setup

The Hero component is looking for: `/images/hamsha-photo.jpeg`

If you name your file differently, update the `src/components/Hero.jsx` file to match your filename.
